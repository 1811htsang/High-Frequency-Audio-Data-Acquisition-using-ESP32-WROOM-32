# High-Frequency-Audio-Data-Acquisition-using-ESP32-WROOM-32
High-performance audio acquisition system on ESP32 sampling at 100ksps via I2S &amp; DMA with zero CPU overhead. Features a Dual-core FreeRTOS architecture isolating Wi-Fi (Core 0) and DSP tasks (Core 1) to ensure real-time stability. Validated signal jitter &lt; 5µs. Designed for precision IoT audio analysis.
